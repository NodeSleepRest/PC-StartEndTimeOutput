# PC-StartEndTimeOutput

あれ？先週の水曜日、いつ退社したんだっけ…？
こんなとき、PCの終了時間を調べれば大体の目星をつけることができます。
PCの終了時間を調べるにはイベントビューアを確認したらよいのですが、
イベントビューアの見かたが分からない…というあなた。

この「PC起動・終了時間リスト作成.ps1」を使えばもう安心。
テキストファイルにして、一日の最初にPCを起動した時刻・最後にPCを終了させた時刻を出力してくれます！
なんとWindows10にも対応してるんです！

PC落としたのが日付変わった後だった〜なんていう無茶な残業をしてる方にはオススメできません。
